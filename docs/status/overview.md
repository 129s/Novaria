# Novaria Overview

## 1. 项目目标

Novaria 当前目标是持续验证并稳固三条核心闭环：

- 可玩闭环：采集、建造、战斗、Boss 挑战与存档回读。
- 技术闭环：固定 Tick、可观测、可回归、可持续迭代。
- 扩展闭环：多人协作与 Mod 扩展在同一规则下保持一致。

## 2. 范围边界

### 2.1 当前范围

- 世界探索与基础生存建造玩法。
- 小规模多人协作体验。
- 可追溯的存档、诊断与发布流程。
- 第三方 Mod 的基础内容扩展能力。

### 2.2 非目标范围

- PVP 与匹配大厅。
- 跨平台联机与跨端发布。
- 大型剧情线与复杂世界事件系统。
- 脚本直接接管底层渲染或网络协议。

## 3. 完成定义（DoD）

1. 可完成一条完整可玩路径，并可存档回读。
2. 目标场景满足稳定帧率预算。
3. 小队协作长时运行保持稳定。
4. 第三方 Mod 扩展通过一致性校验。

## 4. 当前约束

- 目标平台：Windows。
- 主要语言：C++20。
- 图形与输入：SDL3。
- 内容脚本：LuaJIT。
