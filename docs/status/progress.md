# Novaria Progress

## status

- 项目当前按负责人决策从 M6 回退至 M2，可玩闭环重新验收。
- 当前开发主轴：恢复“泰拉式最小可玩闭环”而非继续扩展复杂能力。
- 本轮范围锁定：左键挖右键放、树与静态水体、昼夜背景、简化光照、木剑制作路径。
- 发布闸门以本地脚本执行与人工复核为准。
- 已完成底层改造：调试输入移除、地形生成升级、正式制作链路接入；当前进入体验验收阶段。
