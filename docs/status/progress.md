# Novaria Progress

## status

- 项目当前按负责人决策从 M6 回退至 M2，按新 gameplay 蓝图重建可玩闭环。
- 当前开发主轴：输入与动作模型从“按键直连行为”升级为“左键 ActionBundle、右键 Interaction”。
- 本轮范围锁定：采集与放置分离、工具门槛、放置耗时、世界掉落与接触拾取事件链。
- 阶段策略：先完成无需人工即时介入的逻辑闭环，再集中进行交互与手感验收。
- 发布闸门以自动化测试、日志证据与负责人人工验收共同判定。
