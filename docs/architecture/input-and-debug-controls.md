# 输入与调试控制

## 正式玩家输入（当前）

- `W/A/S/D`：移动角色
- `E`：挖掘角色面向方向的方块
- `R`：放置当前选中材料到角色面向方向
- `1/2`：切换放置材料（`dirt/stone`）

## 兼容调试输入（保留）

- `J`：提交 `jump`
- `K`：提交 `combat.fire_projectile`
- `F1`：发送 `debug.ping`
- `F2`：提交 `world.set_tile`（`0,0,0`）
- `F3`：提交 `world.set_tile`（`0,0,2`）
- `F4`：请求断线
- `F5`：注入心跳
- `F6`：请求重连
- `F7`：采集木材（+5）
- `F8`：采集石材（+5）
- `F9`：建造工作台
- `F10`：制作剑
- `F11`：攻击基础敌人
- `F12`：攻击 Boss
